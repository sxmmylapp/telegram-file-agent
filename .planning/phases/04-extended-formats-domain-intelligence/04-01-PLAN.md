<task type="auto">
  <name>Task 1: Install mammoth and SheetJS, create text extraction service</name>
  <files>package.json, src/services/text-extractor.ts</files>
  <action>
  1. Install dependencies:
     ```bash
     npm install mammoth
     npm install https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz
     ```
     IMPORTANT: Do NOT install xlsx from npm registry (v0.18.5 has CVE-2023-30533). Must use CDN URL.
     Do NOT install @types/mammoth or @types/xlsx -- both libraries bundle their own TypeScript types.

  2. Create `src/services/text-extractor.ts` with two exported functions:

     **extractDocxText(buffer: Buffer): Promise<string>**
     - Use `mammoth.convertToHtml({ buffer })` (NOT extractRawText -- HTML preserves table structure critical for real estate docs)
     - Log mammoth warnings from `result.messages` if any exist
     - Log extracted text length for monitoring
     - Return the HTML string (`result.value`)

     **extractSpreadsheetText(buffer: Buffer, fileName: string): Promise<string>**
     - Use `XLSX.read(buffer)` which auto-detects format (xlsx, csv, etc.)
     - Iterate `workbook.SheetNames`, for each sheet call `XLSX.utils.sheet_to_csv(worksheet, { strip: true, blankrows: false })`
     - Format output as: `## Sheet: {sheetName}\n{csv}` with sheets joined by double newline
     - Log sheet count and total row count for monitoring
     - Log a warning if extracted text exceeds 100KB (potential token cost issue)
     - Return the combined text string

     Both functions accept a `logger: Logger` parameter (pino) for structured logging, matching the existing service pattern.
     Import pattern: `import mammoth from "mammoth"` and `import * as XLSX from "xlsx"` (ESM imports matching project's type:module).
  </action>
  <verify>
  Run `npx tsc --noEmit` -- zero TypeScript errors. Verify mammoth and xlsx appear in package.json dependencies. Verify `src/services/text-extractor.ts` exports both functions.
  </verify>
  <done>
  text-extractor.ts exists with extractDocxText and extractSpreadsheetText exported. Both accept Buffer + Logger. mammoth and xlsx in package.json dependencies.
  </done>
